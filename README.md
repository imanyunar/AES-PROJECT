# üîê Proyek AES dengan S-Box Custom Terbaik

## Deskripsi
Proyek ini menguji Sbox dengan berbagai **properti kriptografi** untuk menentukan S-box terbaik, dan memungkinkan pengguna melakukan **enkripsi dan dekripsi AES** menggunakan S-box tersebut.

Properti yang diuji untuk menentukan S-box terbaik meliputi:
- Nonlinearity (NL)
- Strict Avalanche Criterion (SAC)
- Bit Independence Criterion (BIC-NL, BIC-SAC)
- Differential Uniformity (DU)
- Algebraic Degree (AD)
- Transparency Order (TO)
- Correlation Immunity (CI)

---

## Fitur Utama
1. **Generate S-box otomatis**  
2. **Pengujian kriptografi lengkap**  
3. **Pemilihan S-box terbaik secara otomatis**  
4. **AES encrypt/decrypt menggunakan S-box terbaik**  
5. **Python murni, tanpa library eksternal**
